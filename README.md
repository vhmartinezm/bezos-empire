## Bezos Empire

### Get Started

### Back-end

##### Build Docker image

`docker-compose -f backend/docker-compose.yml build`

##### Start

`docker-compose -f backend/docker-compose.yml up`

### Front-end

##### Install dependencies

`npm install`

##### Run App

`npm start`

Then open http://localhost:3010/ to see the app.
